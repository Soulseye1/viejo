# ☕ Viejo Coffee Menu

A sleek, responsive digital menu for **Viejo Coffee**, built using **React (CRA)**. Designed for in-store display, mobile-first browsing, and seamless deployment on platforms like **Netlify** and **Vercel**.

![Viejo Coffee Logo](public/viejo-logo.png)

---

## 🔧 Project Setup

### 1. Install Dependencies

```bash
npm install
```

### 2. Run Locally

```bash
npm start
```

---

## 🚀 Deploy Instructions

### ✅ Netlify Settings

When deploying to [Netlify](https://netlify.com):

- **Build Command**:  
  ```
  npm run build
  ```

- **Publish Directory**:  
  ```
  build
  ```

- ✅ Add a `_redirects` file inside your `/public` folder with:
  ```
  /*    /index.html   200
  ```

### ✅ Vercel Settings

If deploying to [Vercel](https://vercel.com):

- **Framework Preset**: `Create React App`
- **Build Command**: `npm run build`
- **Output Directory**: `build`
